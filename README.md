# FindPrice
**FindPrice**는 팔고싶은 물건의 중고거래 정보를 알려주는 서비스입니다. 
현재는 당근마켓에서 거래되고 있는 물건 정보를 이용하며, 추후 여러 중고거래 플랫폼의 정보를 추가할 예정입니다.

## 🔧 Tip & Guide

* 1. 물건의 이름을 검색한다.
	1. 이름에는 브랜드와 상품의 종류가 들어가면 좋다.
	2. 추가에정
	
* 2. 얻을 수 있는 정보
	1. 검색 결과의 평균 단가
	2. 검색 결과 중 최고가, 최저가 상품의 정보
    3. 물건들의 가격 분포도

  2. **새로고침하면 오류난다.**
